# LifeFlow App

A simple dashboard I built to help with my night shift schedule. I work 6PM-5AM and needed to see my actual free time.

## Live Link
  https://life-flow-app-gilt.vercel.app/

## What It Does
- Shows my day in colored blocks (work, sleep, free time)
- Tracks small daily activities
- Suggests quick things to do during breaks
- Helps me plan my limited free time better

## Tech Used
- HTML, CSS, JavaScript
- LocalStorage to save activities
- Font Awesome icons
- Deployed on Vercel

## Why I Built It
I was stuck in work-sleep-repeat mode. This app helps me see where my time actually goes and make better use of my free hours. This app has also hellped me become pro acive and motivated me to be more than just a night shift worker.
